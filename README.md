
# QR Code Generator

A simple QR code generator application built with PyQt5.

## Features

- Enter a URL to generate a QR code
- Display the QR code in the GUI
- Save the generated QR code as a PNG image

## Requirements

- Python 3.x
- PyQt5
- qrcode
- Pillow

## Installation

1. Install the required packages:
   ```
   pip install PyQt5 qrcode Pillow
   ```

## Usage

Run the application:
```
python main.py
```

Or run the standalone executable:
```
dist/main.exe
```

1. Enter a URL in the input field.
2. Click "Generate QR Code" to create the QR code.
3. The QR code will be displayed in the window.
4. Click "Save QR Code" to save the image to a file.
